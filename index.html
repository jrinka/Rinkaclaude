How to Play

  1. Click New Round to get a random letter and 12 categories.
  2. Press Start — you have 3 minutes!
  3. Write one answer for each category that starts with the letter shown.
  4. Score 1 point per answer. Bonus: score 2 points if nobody else wrote the same thing!
  5. Most points after 3 rounds wins. 🏆
